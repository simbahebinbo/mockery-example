# Mockery-example


## Get it

```bash
go get github.com/jaytaylor/mockery-example
```

## Run it

```bash
go run main.go
```

### Sample output:

```bash
$ go run main.go
common prefix: {
  Prefix: "2017-01-01"
}
content: {
  Key: "foo-object"
}
```
